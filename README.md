# FitForge-Java-Swing-App

FitForge is a Java Swing desktop application that generates fitness recommendations, BMI calculations, meal plans, and workout plans.

## Important

This project is a **Java Swing desktop app**, not a web app. It cannot be deployed directly to Vercel, Netlify, or similar static hosting services.

## Live project page

A static GitHub Pages landing page is included in the `docs/` folder. To make it available live:

1. Push this repository to GitHub.
2. Go to the repository's GitHub Settings.
3. Enable **GitHub Pages** and publish from the `docs/` folder.

If enabled, the live page will be available at:

`https://harinathreddymatta3955-maker.github.io/FitForge-Java-Swing-App/`

> Note: this page is a project presentation page, not the actual Swing app.

## Run locally

Open the project in an IDE and run `Main.java`, or compile with Java:

```bash
javac src\*.java
java -cp src Main
```

The default app launcher opens the Swing GUI. To use the console interface instead, run:

```bash
java -cp src Main cli
```

## Features

- Desktop Java Swing fitness planner
- BMI and calorie guidance
- Personalized meal and workout plan generation
- Export report to `FitForge_Report.txt`
- Reset form and regenerate plans easily
